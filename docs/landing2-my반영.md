## 1. 헤드라인 섹션

```
💼 15년차 대기업 기획실장이 알려주는
   AI로 기획서 작성 시간 80% 줄이는 법

[서브 카피]
"동료들은 AI로 2시간 만에 끝내는데, 
 나만 사흘째 빈 PPT 앞에서 고민하고 계신가요?"

✓ 완벽한 기획서가 아닌, '통과되는' 기획서를 빠르게
✓ 네이버 기획실장 노하우 × ChatGPT 실전 활용
✓ 기획서/사업계획서/제안서 3일 → 반나절로 단축
```

**시각 요소**
- Before: 야근하는 직장인 / After: 정시 퇴근
- 타이머: "72시간 → 6시간" 강조

## 2. 문제 인식 섹션

### 🚨 이런 경험, 있으시죠?

**탭 1: "나만 시간이 없다"**
```
□ 기획서 하나 만드는데 일주일, 정작 핵심 업무는 손도 못 댐
□ PPT 디자인에만 이틀을 쏟고, 정작 내용은 부실
□ 투자 유치/공모사업 마감은 다가오는데 초안도 없음

→ 🎯 해결책: 
   스토리라인 2시간 → AI 초안 30분 → 다듬기 2시간
   하루 안에 검토 가능한 초안 완성
```

**탭 2: "AI 써봤는데 별로더라"**
```
□ ChatGPT에 "사업계획서 써줘" → 뻔한 내용만
□ 프롬프트 100개 암기했는데, 막상 실무엔 안 맞음
□ AI 초안 고치다가 차라리 처음부터 쓰는 게 빠를 뻔

→ 🎯 해결책:
   "좋은 프롬프트" 암기가 아닌
   "15년차 기획자의 프로세스"를 AI에 학습시키는 법
```

**탭 3: "동료들은 이미 쓰고 있다"**
```
□ 팀 회의에서 "AI로 빠르게 작업했어요" 말이 자주 들림
□ 나만 여전히 빈 PPT 앞에서 막막함
□ "AI 활용 역량" 평가 항목 생기면 어떡하지...

→ 🎯 해결책:
   이 강의는 "AI 전문가" 되려는 게 아닙니다
   동료들처럼 "업무에 AI 쓸 줄 아는" 수준이면 충분합니다
```

```

• 이미 AI 기획 자동화 시스템 갖추신 분
```

```
   잘못된 습관 없이 '프로세스 중심' 활용법을
   처음부터 배울 수 있습니다.
   수강생 58%가 AI 초보자였습니다.

Q. 온라인 강의도 있나요?
A. 현재는 오프라인 집중 코스만 운영 중입니다.
   실습 피드백과 즉각적인 질의응답이
   학습 효과를 3배 높이기 때문입니다.
   (온라인 과정은 2026년 하반기 오픈 예정)

───────────────────────────────────

Q. 정말 3일 → 반나절로 단축되나요?
A. "완벽한 최종본"은 아닙니다.
   "검토를 시작할 수 있는 70-80점 초안"이
   5-6시간 내 완성됩니다.
```
