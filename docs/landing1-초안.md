## 2. 문제 인식 섹션 (공감대 형성)

### 제목
```
🚨 이런 상황, 익숙하지 않으신가요?
```

### 3개 탭 구조 (클릭/토글)

#### 탭 1: "나만 뒤처지는 것 같아요" (Sociotechnical Blindness 자극)
```
□ 팀 회의에서 "AI로 작업했어요"라는 말이 자주 들리는데, 
  나만 여전히 밤샘 작업 중

□ 후배는 ChatGPT로 기획서를 후다닥 만드는데,
  나는 그게 어떻게 가능한지 모르겠음

□ "AI 시대에 뒤처지면 안 되는데..." 막연한 불안감만

→ 🎯 해결책: 이 강의는 "AI 전문가"가 되려는 게 아닙니다.
   동료들처럼 "업무에 AI를 쓸 줄 아는" 수준이면 충분합니다.
```

#### 탭 2: "시간은 없고, 완벽해야 하고" (시간 압박)
```
□ 투자 유치 앞두고 사업계획서 급하게 만들어야 함
□ 공모사업 마감은 다음 주인데 아직 초안도 없음
□ 기획서 작성에만 3일, 정작 핵심 업무는 손도 못 댐

→ 🎯 해결책: 완벽한 최종본이 아닌, "검토 시작할 수 있는" 
   70점짜리 초안을 2시간 만에 만드는 법
```

#### 탭 3: "ChatGPT 써봤는데 별로더라" (기대 실패 경험)
```
□ ChatGPT에 "사업계획서 써줘" 했더니 뻔한 내용만
□ 프롬프트 100개 강의 들었는데, 막상 실무엔 안 맞음
□ 결국 처음부터 다시 작성... 시간만 낭비

→ 🎯 해결책: "좋은 프롬프트"보다 중요한 건 
   "실무 흐름에 맞는 작업 시스템"입니다.
```
