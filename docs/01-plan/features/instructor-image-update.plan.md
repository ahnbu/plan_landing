# 강사 이미지 반영 계획서

> 버전: 1.0.0 | 작성일: 2026-02-10 | 상태: 초안

## 1. 요약 (Executive Summary)
`landing_gemini.html` 파일의 강사 소개 섹션에 배경이 제거된 실제 강사 사진(`강사 강연 사진_2차_정장_bg제거.png`)을 반영하여 신뢰도와 시각적 완성도를 높입니다.

## 2. 목표 (Goals and Objectives)
- 기존 플레이스홀더 또는 잘못된 경로의 이미지를 실제 파일로 교체
- 배경 제거된 PNG 파일의 특성을 살려 자연스러운 합성 효과 구현
- 이미지 경로 및 확장자(`jpg` -> `png`) 수정

## 3. 범위 (Scope)
### 포함 범위 (In Scope)
- `docs/강사 강연 사진_2차_정장_bg제거.png` 파일을 프로젝트 루트로 복사 (또는 적절한 경로 지정)
- `landing_gemini.html` 내 `<img>` 태그의 `src` 속성 수정
- 이미지 컨테이너 스타일(배경색 등) 최적화

### 제외 범위 (Out of Scope)
- 이미지 보정 및 편집
- 강사 프로필 텍스트 수정

## 4. 성공 기준 (Success Criteria)
| 기준 | 지표 | 목표 |
|-----------|--------|--------|
| 이미지 표시 | 로딩 여부 | 브라우저에서 강사 사진이 깨지지 않고 표시됨 |
| 투명도 적용 | 시각 효과 | 배경 제거된 영역에 HTML의 배경 그라디언트가 비침 |
| 정렬 | 레이아웃 | 이미지가 컨테이너 내에서 올바른 비율(aspect-ratio)로 표시됨 |

## 6. 리스크 (Risks)
| 리스크 | 영향 | 완화 방안 |
|------|--------|------------|
| 파일 경로 불일치 | 이미지 미표시 | 상대 경로를 정확히 지정하고 `onerror` 핸들러 유지 |
| 대용량 이미지 | 페이지 로딩 저하 | 필요시 이미지 최적화 권장 (현재는 원본 반영) |
