# 강사 이미지 반영 설계서

> 버전: 1.0.0 | 작성일: 2026-02-10 | 상태: 초안

## 1. 개요 (Overview)
본 설계서는 `landing_gemini.html`에 배경 제거된 강사 이미지를 적용하기 위한 상세 절차를 정의합니다.

## 2. 자산 관리 (Asset Management)
- **원본 파일**: `docs/강사 강연 사진_2차_정장_bg제거.png`
- **복사 대상**: `./강사_강연_사진_정장_bg제거.png` (파일명 간소화 및 공백 관리 용이성 고려)

## 3. 코드 수정 사항 (Code Modification)

### 3.1 이미지 태그 수정
- **파일 경로**: `landing_gemini.html`
- **수정 위치**: 215번 라인 부근
- **수정 내용**:
    - `src` 속성을 복사된 파일명으로 변경.
    - `onerror` 처리 시 대체 이미지 유지.
    - `object-top` 설정을 통해 인물의 상반신 위주로 노출되도록 조정.

### 3.2 스타일 보정 (선택 사항)
- 현재 이미지가 들어갈 컨테이너(`bg-gradient-to-b from-gray-800 to-black`)가 배경 제거된 인물 뒤에서 은은한 배경 역할을 하도록 설계됨. 별도 스타일 변경 없이 유지.

## 4. 실행 절차 (Execution Steps)
1. `docs/` 폴더에서 원본 이미지를 루트 디렉터리로 복사.
2. `landing_gemini.html`의 `src` 경로를 업데이트.
3. 파일명에 포함된 공백을 처리하기 위해 안전한 파일명으로 리네임 고려.

## 5. 테스트 계획 (Test Plan)
- 이미지 경로가 정확한지 확인.
- 브라우저 개발자 도구에서 404 에러가 발생하지 않는지 확인.
